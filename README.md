# EchoLearning - 英语精听学习平台

基于「四阶段学习法」的英语听力训练应用。

## 四阶段学习法

1. **素材筛选** - 选择感兴趣的内容（20分钟新闻/访谈）
2. **盲听听写** - 训练耳朵对单词的敏感度
3. **深度理解** - 消除所有知识盲点
4. **极限背诵** - 形成语感和语言习惯

## 技术栈

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Zustand (状态管理)
- Browser TTS (语音播放)

## 功能

- 素材库管理（上传、筛选、分类）
- 盲听播放器（语速调节、进度控制）
- 听写练习区
- 词汇本
- 学习笔记
- 背诵模式（段落练习、进度追踪）
- 数据持久化（LocalStorage）
